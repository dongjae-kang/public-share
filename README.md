# public-share

A staging area for files and assets that need a **public link for external sharing**.

- Drop standalone HTML, PDFs, images, or any artifact here when you need a shareable URL.
- Core/source work lives in its own (often private) repos. This repo holds only what is intentionally public.
- With GitHub Pages enabled, files are reachable at `https://dongjae-kang.github.io/public-share/<path>`.

Not topic-specific. General-purpose link surface.
